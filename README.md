# RESOLVER

A social betting app where two friends lock real USDC into a bet, route funds through a DeFi yield vault, and the winner collects the full pot plus any yield generated.

## Setup

1. `cd backend && npm install`
2. Copy `.env.example` to `.env` and add `LIFI_API_KEY`
3. `node server.js`
4. Visit http://localhost:3001/health
5. Visit http://localhost:3001/earn/best-vault to verify LI.FI connection

API key signup: https://portal.li.fi/signup
